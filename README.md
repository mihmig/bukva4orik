### bukva4orik_bot
Игровой телеграм-бот по мотивам игры Wordle

### Идеи

* Старт только админом (Команда /admin@bukva4orik_bot)
* Глобальный рейтинг и рейтинг в группе (при блокировке/удалении бота - удялять записи из рейтинга)
* Вечерние чемпионаты (победитель может сказать что-нибудь через бота, согласованное с администрацией)
* Тематические игры (админы могут загадывать слова)
### Ачивки
Разработать систему ачивок:
5,6,7 букв
